# EcoPulse: Enterprise Sustainability Intelligence

![EcoPulse Banner](https://images.unsplash.com/photo-1473341304170-971dccb5ac1e?auto=format&fit=crop&q=80&w=2070)

### Precision AI Analytics for Net-Zero Enterprise Growth

EcoPulse is a premium, AI-driven sustainability decision platform designed to transform complex industrial data into actionable ESG intelligence. Built for modern enterprises, EcoPulse empowers organizations to achieve ambitious decarbonization targets while identifying high-yield capital recovery opportunities.

---

## 🚀 Key Modules

### 📊 Executive Analytics
A high-fidelity dashboard providing real-time telemetry on energy consumption, carbon inventory, and industry benchmarks. Features dynamic KPI mapping and automated sustainability rating audits.

### 🧪 ROI Strategic Simulator
An advanced scenario planning environment that allows leaders to model the fiscal and environmental impact of operational shifts (Solar Integration, Lighting Retrofits, etc.) with neural accuracy before committing capital.

### 🤖 ESG AI Advisor
A sophisticated strategic assistant powered by precision NLP. The advisor provides instant insights into global ESG frameworks, identifies operational anomalies, and recommends targeted carbon abatement measures.

### 💰 Capital Recovery Finder
An intelligent matching engine that cross-references enterprise activity profiles with global incentive marketplaces, identifies eligible green grants, and facilitates rapid capital recovery.

### 📄 Smart Audit Engine
Automated document ingestion and OCR-powered data extraction. Securely processes utility invoices and operational logs to build a verified baseline for ESG reporting.

## 🏗️ Technical Architecture

EcoPulse is built on a high-performance, modular stack designed for scalability and secure data processing.

### **Frontend: Premium Interface**
- **Framework**: React 18 + Vite
- **Animation**: Framer Motion for cinematic transitions
- **Visualization**: Recharts & D3.js for precision telemetry
- **Design System**: Custom "Emerald & Slate" enterprise identity with Adaptive Glassmorphism

### **Backend: Intelligence Core**
- **Framework**: FastAPI (High-performance Python 3.10+)
- **Vector Database**: ChromaDB for intelligent policy & incentive retrieval
- **Database**: PostgreSQL for structured enterprise data & audit logs
- **AI/ML**: 
  - HuggingFace Transformers for NLP-driven strategy synthesis
  - All-MiniLM-L6-v2 for semantic vector embeddings
  - Intelligent recommendation heuristic engine

## 🔧 Deployment & Setup

### **1. Prerequisites**
- Python 3.10+
- Node.js 18+
- Docker & Docker Compose (Optional for containerized deployment)

### **2. Installation**

```bash
# Clone the repository
git clone https://github.com/CNI-GenAI-Hackathon2025/track-3-agenticai-angelina-2206.git
cd EcoPulse_project

# Set up Backend
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt

# Set up Frontend
cd frontend
npm install
cd ..
```

### **3. Launch the Platform**
EcoPulse includes a unified bootstrapper to launch both services simultaneously:

```bash
python run.py
```

- **Enterprise Dashboard**: [http://localhost:8501](http://localhost:8501)
- **API Specification**: [http://localhost:8000/docs](http://localhost:8000/docs)
